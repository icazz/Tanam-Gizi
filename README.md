# 🌱 TanamGizi — Aplikasi Rekomendasi Menu Gizi 1000 HPK

Aplikasi mobile rekomendasi menu berbasis pangan lokal untuk ibu hamil dan menyusui dalam mendukung pencegahan stunting pada 1000 Hari Pertama Kehidupan.

---

## 📁 Struktur Repositori

```
tanam-gizi/
├── mobile/          → Anggota 1: Aplikasi Mobile (Flutter/React Native)
├── backend/         → Anggota 2: API Server & Database (FastAPI)
├── ai-engine/       → Anggota 3: AI/ML Engine & Recommendation System (Python)
├── data/            → Dataset bersama (DKBM, AKG, Resep)
└── docs/            → Dokumentasi teknis tim
```

---

## 👥 Pembagian Tugas Tim

| Anggota | Role | Direktori Utama |
